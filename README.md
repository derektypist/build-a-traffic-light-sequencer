# Build a Traffic Light Sequencer

Simulate configurable traffic light cycles and log anomalies.

## Technologies

Uses JavaScript with for or while loops.

## Deployment

Copy `script.js` into the terminal of your choice.  If using terminal, use the command `node script.js` to run the code.
